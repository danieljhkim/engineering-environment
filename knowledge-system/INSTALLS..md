# Knowledge System — Installs

This document lists **only the required and recommended tooling** for the knowledge system.

---

## Required

### Obsidian
Primary interface for the vault.

- Download: https://obsidian.md
- Platforms: macOS, Linux, Windows

No account or cloud sync is required.

---

## Recommended

### Dataview (Obsidian Community Plugin)
Used for dashboards, project tables, and task queries.

**Install**
1. Obsidian → Settings
2. Community plugins → Browse
3. Search **Dataview**
4. Install and Enable
